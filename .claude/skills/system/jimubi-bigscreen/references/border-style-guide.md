# JDragBorder 边框样式速查

> 按需加载：用户问"用哪种边框"、"边框选哪个"、"边框有什么区别"时读本文件。

## 13 种样式特性一览

| variant | 视觉特征 | 适用场景 |
|---------|---------|---------|
| **1** | 圆角矩形 + 四角小刻度标记，单线简洁 | 通用首选，KPI 卡片、图表容器，适配任何主题 |
| **2** | 极简纯矩形单线，无任何装饰 | 内容为主的数据区域，不抢视觉焦点，适合大数据表、地图 |
| **3** | 双层/三层同心矩形边框，有层次感 | 重要模块突出展示，核心 KPI 区域、指挥中心中央区 |
| **4** | 仅四角 L 形角标，无连接线 | 轻量点缀，空间有限时使用，装饰性大于框架性 |
| **5** | 顶部+底部全宽横线 + 左侧斜切角，不对称设计 | 工业风/军事风大屏，设备监控、生产线仪表盘 |
| **6** | 简洁矩形 + 底部微开口，轻量感 | 表格类容器，比 variant 2 多一点设计感 |
| **7** | 内发光/辉光边框，红色晕染强烈 | 告警/预警状态展示，重点数据需要吸引眼球时 |
| **8** | 细线矩形，极简 | 多图并列时统一风格，密集布局中减少干扰 |
| **9** | 四角阶梯式缺口（机械感切角） | 工业/制造/IoT 大屏，机械设备监控，赛博朋克风格 |
| **10** | 超强外发光，红色光晕浓郁 | 高危告警/红色主题，数据异常时的状态容器，视觉冲击最强 |
| **11** | 顶部居中胶囊形标题栏 + 两侧铆钉 + 角部机械件，最复杂 | 主图表/核心区域专用，适合有标题的图表区，科技感最强，宜单独使用 |
| **12** | 斜角切边 + 角部折线装饰，有棱角感 | 警务/安防/军事风格大屏，需要棱角感而非圆角的场景 |
| **13** | 大圆角矩形 + 顶部三小圆点，类面板窗口 | 现代简约风，适合医疗/教育/商务主题，风格柔和 |

## 主题 → variant 速查

| 主题风格 | 推荐 variant |
|---------|-------------|
| 极简 / 商务 | 2、8、13 |
| 通用科技 | 1、6 |
| 重点突出 / 核心区 | 3、11 |
| 工业 / 机械 / IoT | 5、9、12 |
| 告警 / 红色主题 | 7、10 |
| 轻量装饰 | 4 |
| 赛博朋克 | 9、11、12 |

## 使用原则

- 同一大屏不超过 3 种 variant 混用，保持视觉统一
- variant 7 / 10（发光辉光）只在告警/异常状态使用，不做常规容器
- variant 11（最复杂）适合单独作为核心区大图容器，不宜密集平铺
- variant 4 面积较小时效果最佳，组件尺寸过大时角标会显得稀疏
