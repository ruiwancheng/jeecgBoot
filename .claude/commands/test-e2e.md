# /test-e2e <项目名> [功能名]

浏览器全流程测试。发现问题后自动尝试修复。

## 流程

### 1. 加载领域知识
使用 `test-e2e` 技能获取：错误诊断与修复映射表、前端组件引用、Playwright 运行命令。

### 2. 找测试
检查 `harness/e2e/<项目名>/<功能名>.spec.ts`，没有则自动生成。

### 3. 跑测试
Playwright 浏览器测试。

### 4. 通过 → 告知
```
<功能名> E2E 测试：全部通过。
```

### 5. 失败 → 自动修复 → 重测
按技能中的错误诊断映射表匹配报错类型，执行对应修复方案。修复后重跑，最多尝试 3 次。修复失败则告知需要技术人员协助。
