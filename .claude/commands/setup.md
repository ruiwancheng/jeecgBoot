# /setup <项目名>

项目模块代码写好后，一键完成数据库初始化 + 编译 + 重启。

## 执行步骤

### 1. 加载领域知识
使用 `setup` 技能获取：SQL 文件路径模式、数据库连接信息、Maven 编译命令、依赖关系。

### 2. 执行 SQL 初始化
按技能中的 SQL 路径模式逐个执行建表脚本。

### 3. 编译安装模块
按技能中的 Maven 编译命令编译安装。

### 4. 重启后端
执行 `/restart-backend`。

### 5. 告知
新菜单在前端刷新后可见。
