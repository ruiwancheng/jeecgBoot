# /harness-check

Harness 工程健康度诊断。7 轴评估 + 演进检查 + 改进建议。

## 流程

### 1. 加载领域知识
使用 `harness-check` 技能获取：完整文件清单、8 轴评分细则（含命令与技能边界检查、交叉验证、代码结构健康）、演进检查方法。

### 2. 文件完整性检查
按技能中的文件清单逐项检查是否存在且内容完整。

### 3. 8 轴诊断
按技能中的评分细则逐轴打分（每轴 0-5 分）。第 8 轴需调用 MCP graph tools。

### 4. 演进检查
逐 Rules 问"还编码了有效假设吗？"，逐 Commands 问"还在使用吗？"

### 5. 输出报告
**铁律：必须输出改进建议。只评分不给建议 = 未完成。**

报告后询问：
```
修复哪些？(1,2,3 / all / none)
```
确认的项自动落地。
