# /start

启动本地开发环境。首次自动初始化数据库，后续秒级唤醒。

## 流程

使用 `local-dev` 技能，按 5 步自动执行：

1. 检查 MySQL、Redis、Java、Node 是否就绪（未运行则自动启动）
2. 数据库初始化（首次导入表结构，幂等跳过）
3. 安装前端依赖（仅首次）
4. 启动后端（端口 8080，DevTools 热重载）
5. 启动前端（端口 3100，Vite 热更新）

**用户只需等 30 秒看到"就绪"提示，打开 http://localhost:3100 即可。**

## 使用方式

```
用户：/start
AI  ：🔍 环境检查中... → 🗄️ 数据库就绪 → 🚀 后端启动 → 🎨 前端启动 → ✅ 就绪！
```

## 停止环境

```
用户：帮我停止本地环境
AI  ：停止后端 + 停止前端
```

## 部署到服务端

本地验证完 → `git commit && git push` → 到 http://100.122.125.106:3101 点击"开始部署"
