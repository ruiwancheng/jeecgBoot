# /test-frontend <项目名> <功能名>

前端快速自测，不打开浏览器，秒级出结果。适合日常开发中频繁使用。

## 流程

### 1. 加载领域知识
使用 `test-frontend` 技能获取：组件完整性检查模式、路由检查规范、TypeScript 类型检查命令、构建验证命令。

### 2. 组件完整性
检查页面文件引用的子组件都存在。

### 3. 路由检查
验证路由文件指向的组件是否存在。

### 4. 类型检查
运行 TypeScript 类型检查。

### 5. 构建验证
运行生产构建。

### 6. 输出
每项通过/失败，失败时列出具体文件和原因。
