# /new-project <项目名>

创建新项目，自动完成所有初始化。

## 自动创建内容

| 内容 | 说明 |
|------|------|
| 代码目录 | 后端 Maven 模块 |
| 前端目录 | 前端页面 |
| 前端路由 | 路由注册 |
| 角色 | 编码同项目名 |
| 用户 | 默认密码见技能配置 |
| 当前项目 | 激活 |

## 执行步骤

### 1. 加载领域知识
使用 `new-project` 技能获取：Maven 模板路径、占位符替换规则、Java 包路径、SQL 模板、TypeScript 路由模板、.manifest.yml 模板、激活文件写法。

### 2. 复制模板并替换占位符
按技能中的模板路径复制并执行占位符替换。

### 3. 注册 Maven
按技能中的注册位置（两处 pom.xml）添加模块和依赖。

### 4. 创建角色和用户
按技能中的 SQL 模板执行。

### 5. 创建前端路由
按技能中的 TypeScript 路由模板生成路由文件。

### 6. 激活项目
按技能中的激活文件格式写入当前项目。

### 7. 告知
项目已创建，告知用户账号和密码。
