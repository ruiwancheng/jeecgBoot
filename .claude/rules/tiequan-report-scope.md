# 铁拳团审计报告归档规则

## 1. 目的

统一铁拳团审计产出存放位置，避免报告散落在 `hermes/audits/`、`hermes/reports/` 等多个目录。

## 2. 归档目录

所有铁拳团审计产出必须统一放在：

```
hermes/tiequan/YYYY-MM-DD/<模块名>/
```

例如：

```
hermes/tiequan/2026-07-13/warehouse-zone-shelf-location/
hermes/tiequan/2026-07-13/mes-customer/
```

## 4. 禁止行为

- 禁止将铁拳团报告放到 `hermes/audits/` 目录。
- 禁止用中文日期命名目录（如 `2026年07月13日`）。
- 禁止专项报告文件名缺失 `01_` 等序号前缀。
- 禁止只生成总报告而不生成 10 份专项报告。

## 5. 规则生效

本规则由赤兔与老大共同确认，自 2026-07-13 起执行。后续铁拳团任务必须按此规则归档。
